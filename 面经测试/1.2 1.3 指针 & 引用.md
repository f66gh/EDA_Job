# 🧪 **【指针 & 引用深度理解测试卷】**

### ——测测你是不是真的“懂了”

------

| 表达式 | 类型     | 是什么                 |
| ------ | -------- | ---------------------- |
| a      | int      | 值                     |
| &a     | int*     | a 的地址               |
| p      | int*     | 存的是一个地址         |
| *p     | int      | p 指向的值             |
| int* p | 类型声明 | p 是指针               |
| int& r | 类型声明 | r 是引用               |
| &r     | 操作符   | r 的地址（= a 的地址） |
